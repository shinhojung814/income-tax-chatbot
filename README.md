# 🤖 소득세 챗봇 (Income Tax Chatbot)

한국 **소득세 관련 질문**에 답변하는 AI 챗봇입니다.  
LangChain, OpenAI, Pinecone, Streamlit을 활용하여 구축되었습니다.

---

## 📘 프로젝트 개요

이 프로젝트는 **한국 소득세 법률 문서**를 기반으로,  
사용자의 자연어 질문을 이해하고 관련 법 조항을 바탕으로 답변을 제공합니다.

LLM과 RAG(Retrieval-Augmented Generation) 기법을 활용하여,  
사용자가 입력한 질문에 대해 **벡터 검색 기반의 컨텍스트 보강 답변**을 생성합니다.

---

## 🧩 주요 기술 스택

| 구분       | 기술                                             | 설명                                      |
| ---------- | ------------------------------------------------ | ----------------------------------------- |
| 프론트엔드 | **Streamlit**                                    | 간단한 UI로 대화형 챗봇 인터페이스 제공   |
| 백엔드     | **LangChain**                                    | LLM 체인 구성 및 데이터 흐름 제어         |
| 임베딩     | **OpenAI Embeddings (`text-embedding-3-large`)** | 텍스트를 벡터로 변환                      |
| LLM        | **GPT-4o (OpenAI)**                              | 답변 생성용 대형 언어 모델                |
| 벡터 DB    | **Pinecone**                                     | 소득세 관련 문서의 임베딩을 저장하고 검색 |
| 환경 변수  | **dotenv**                                       | `.env` 파일을 통한 API Key 관리           |
